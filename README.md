#  CSPM Tool (Cloud Security Posture Management)

##  Project Description
This project is a Cloud Security Posture Management (CSPM) tool that continuously scans cloud resources, detects misconfigurations, and automatically remediates them.

This is a simulated implementation inspired by real cloud platforms like AWS, Azure, and GCP.

---

##  Features
- Continuous scanning of resources
-  Detection of security violations
-  Automatic remediation (self-healing)
-  Logging of all actions
-  Continuous monitoring loop

---

##  How it Works

1. Scan resources
2. Detect violations
3. Apply fixes automatically
4. Log the actions
5. Repeat continuously

---

##  Technologies Used
- Python
- File handling (logging)
